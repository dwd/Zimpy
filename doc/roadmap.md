## Quick-Implementation Roadmap (xmpp_stone)

Ordered from easiest to more involved, based on what the library already exposes.

1) Presence controls (set status/show, away/dnd, go invisible) ✅
2) Chat state notifications (typing/paused/inactive) ✅
3) Basic roster actions (add/remove contact, approve/deny subscription)
4) Ping/keepalive status indicator ✅
5) VCard profile fetch (avatar, nickname) ✅ (avatar only)
6) Bookmarks 2 (show saved rooms) ✅
7) Message carbons (sync messages across clients) ✅
8) Message archive (MAM) basic history fetch ✅
9) Stream management toggles (resume, ack status)
10) Privacy lists (block/unblock JIDs)
11) Server features discovery view
